# Archimmersion Website (Eleventy)

Basic informational website scaffold using [Eleventy](https://www.11ty.dev/).

## Quick start

1. Install dependencies:
   - `npm install`
2. Run development server:
   - `npm run dev`
3. Build static site:
   - `npm run build`

## Project structure

- `src/` site source files
- `src/_includes/layouts/` shared Nunjucks layouts
- `src/_data/` global data files
- `src/css/` stylesheets
- `.eleventy.js` Eleventy config
- `_site/` generated output (after build)

## Next edits you will likely make

- Replace placeholder company text in `src/_data/site.js`
- Add/edit pages in `src/*.njk`
- Expand styles in `src/css/site.css`
